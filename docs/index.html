ON TO 2050 Indicators Dashboard

This is a simple dashboard for displaying the latest data from CMAP's ON TO 2050 indicators. The source data (in CSV format) and code can be found on GitHub. Please submit any questions or feedback via this form.

Select an indicator to display from the list of ON TO 2050 chapters below.

(Please be aware that a small number of secondary indicators have been excluded because they currently only have a single observation and no targets, and are therefore not yet suitable for graphing.)

  • Community
  • Prosperity
  • Environment
  • Governance
  • Mobility
  • Inclusive Growth
